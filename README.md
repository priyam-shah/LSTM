# LSTM Neural Network

LSTM built using the Keras Python package to predict stock price.

## Requirements
Install below requirements to make sure correct versions of libraries are being used.

* numpy 1.22.0
* pandas 1.3.5
* scikit-learn 1.0.2
* tensorflow 
* keras 0.7.0
* matplotlib 3.5.1
* matplotlib-inline 0.1.3
